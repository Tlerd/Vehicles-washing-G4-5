# User Stories: AutoWash Pro v2

## Customer and Guest

- As a customer, I want to register with Phone OTP or Google Sign-In, so that I can use one secure linked account.
- As a guest, I want to book with a verified phone number, so that I can reserve a wash without creating an account.
- As a customer, I want to choose a branch, services, a 15-minute time slot, a vehicle, and review pricing before payment, so that I can make an informed booking.
- As a booking user, I want to pay the required VNPAY deposit, so that my slot is automatically confirmed after payment verification.
- As a customer or guest, I want to check in myself and confirm completed work, so that service completion is visible and trustworthy.
- As a customer, I want loyalty points, tiers, vouchers, and feedback after completed service, so that my return visits are rewarded.

## Counter Staff

- As a staff member, I want to see checked-in work by bay and start/finish service, so that operational work is coordinated without controlling customer confirmation.
- As a staff member, I want to correct a vehicle size with a recorded reason, so that the counter balance is accurate.

## Administrator

- As an administrator, I want to manage services, combos, staff, guests, customers, and tiers, so that operations are configurable.
- As an administrator, I want to reschedule bookings with an audited reason and customer notification, so that exceptions are handled transparently.
- As an administrator, I want revenue and audit reports based on immutable booking snapshots, so that historic figures remain correct.
